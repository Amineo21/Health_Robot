# AGENTS.md

## What This Repo Is

- Monorepo:
- `frontend/health-robot-front`: TanStack Start (Vite) React app.
- `backend/`: FastAPI service that subscribes/publishes MQTT topics.
- `infra/`: `docker-compose.yml` wiring Mosquitto + backend + frontend (+ a MySQL container).

## Fastest "It Works" Run (Recommended)

- `docker compose -f infra/docker-compose.yml up --build`
- Exposed ports (per `infra/docker-compose.yml`):
- Frontend `http://localhost:3000`
- Backend `http://localhost:4000` (has `/` and `/health`)
- MQTT: `1883` (tcp) and `9001` (websockets)

### Docker MySQL Gotcha

- `infra/docker-compose.yml` includes `robot_db` (`mysql:8`). MySQL requires one of: `MYSQL_ROOT_PASSWORD`, `MYSQL_ALLOW_EMPTY_PASSWORD`, or `MYSQL_RANDOM_ROOT_PASSWORD`.
- `infra/.env` currently defines `DB_*` vars but not the required `MYSQL_*` ones, so `robot_db` may fail to start until you add one.

## Frontend (Local)

- Workdir: `frontend/health-robot-front/`
- Install: `npm ci`
- Dev: `npm run dev` (forces port `3000`)
- Test (CI-style, single run): `npm run test`
- Focused test: `npx vitest run src/path/to/test.test.tsx -t "name"`

### Generated Route Tree

- `src/router.tsx` imports `./routeTree.gen`.
- `routeTree.gen.ts` is generated by TanStack tooling during dev/build and is not committed; treat it as read-only.

## Backend (Local)

- Workdir: `backend/`
- Install: `pip install -r requirements.txt`
- Run: `uvicorn app.main:app --reload --port 4000`

### MQTT Environment Gotcha

- Backend reads MQTT config from env vars in `app/core/config.py`: `MQTT_HOST`, `MQTT_PORT`, etc.
- `infra/.env` currently sets `MQTT_BROKER=...` (not used by the backend code). If MQTT connectivity is broken in Docker, set `MQTT_HOST=mosquitto` (and `MQTT_PORT=1883`) in `infra/.env`.

### Python Version Mismatch

- `backend/pyproject.toml` declares `requires-python = ">=3.12"`, but `backend/Dockerfile` uses `python:3.11-slim`.
- If you need a reliable runtime baseline, prefer the Dockerfile/`requirements.txt` unless the repo is updated to align versions.

## CI Reality Check
