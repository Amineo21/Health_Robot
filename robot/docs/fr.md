# Documentation en francais

Recettes pour le Yahboom ROSMASTER M3 Pro sous ROS 2 Humble.

Note: cette documentation est ecrite sans accents, pour rester lisible dans
tous les terminaux et faciliter le copier-coller.

Ordre de lecture conseille:

1. **[Demarrage](fr/demarrage.html)** - se connecter au robot et lancer la
   couche materielle.
2. **[Navigation](fr/navigation.html)** - construire une carte, puis naviguer
   en autonomie.
3. **[Bras](fr/bras.html)** - bouger le bras 6 axes et ramasser des objets.
4. **[Integration](fr/integration.html)** - faire tourner cartographie,
   navigation, vision et bras ensemble.
5. **[Tableau de bord](fr/tableau-de-bord.html)** - surveiller le robot depuis
   un navigateur, Foxglove ou RViz.
6. **[Recettes](fr/recettes.html)** - extraits de code a copier-coller.
7. **[Depannage](fr/depannage.html)** - corriger les erreurs frequentes.
