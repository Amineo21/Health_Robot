# English documentation

Recipes for the Yahboom ROSMASTER M3 Pro on ROS 2 Humble.

Suggested reading order:

1. **[Getting started](en/getting-started.html)** - connect to the robot and
   start the hardware layer.
2. **[Navigation](en/navigation.html)** - build a map, then drive autonomously.
3. **[Arm control](en/arm.html)** - move the 6-axis arm and pick objects.
4. **[Integration](en/integration.html)** - run mapping, navigation, vision
   and the arm together.
5. **[Dashboard](en/dashboard.html)** - monitor the robot from a browser,
   Foxglove or RViz.
6. **[Recipes](en/recipes.html)** - copy-paste code snippets.
7. **[Troubleshooting](en/troubleshooting.html)** - fix the common errors.
